## 🚀 How to Scan for IPs

1. Open the CF-IP-Scanner application.
2. The main window displays a list of Cloudflare subnets.
3. Click the Start Scan button.
4. Watch the progress bar as the tool tests different addresses.
5. The software checks latency and download speeds for each tested IP.
6. Once the process completes, the app sorts the results.
7. The fastest IP addresses appear at the top of the list.

## 📊 Using Your Results

After the scan finishes, you have a list of high-performance IP addresses. You can use these addresses in your proxy tools:

1. Select an IP address from the results list in the app.
2. Copy the IP address to your clipboard.
3. Open your proxy client, such as Xray or V2Ray.
4. Locate the configuration settings for your server.
5. Paste the new IP address into the Address or Host field.
6. Save your settings and reconnect your proxy client.

Using a fresh, clean IP address often solves connection issues and slow speeds caused by congested networks.

## ⚙️ Configuration Settings

The Settings menu lets you change how the scanner works:

*   **Scan Thread Count:** Choose how many tasks the app performs at once. A higher number speeds up the scan but uses more computer resources.
*   **Timeout Limit:** Set the maximum time the app waits for a response from an IP. A lower value makes the scan faster but might ignore potentially good addresses.
*   **Export Results:** Save your scan findings to a text file for future reference. This is helpful if you want to test different IPs later.

## 🔍 Understanding the Interface

The interface provides useful data:

*   **Status Indicators:** Icons show you if an IP address is currently unreachable or active.
*   **Latency Column:** Displays the delay in milliseconds. Lower numbers mean a faster connection.
*   **Success Rate:** Shows how often the IP address responds without errors. Higher percentages indicate a more reliable path.
*   **Analytics Tab:** View charts and graphs that explain your connection history. This helps you track performance over several days.

## 🛡️ Privacy and Security

CF-IP-Scanner runs locally on your machine. The software does not send your data to external servers. Your IP results remain private within your Windows installation. ## 📈 Troubleshooting

If the scanner does not return results, check these items:

1.  **Check your internet connection:** Ensure your computer has a stable link to the web.
2.  **Firewall settings:** Windows Defender or third-party antivirus software might block the scanner. Ensure the app has permission to access the network.
3.  **Administrator rights:** Running the app as an administrator can resolve restricted network access issues. Right-click the app icon and select Run as administrator.
4.  **Restart:** A simple restart of the application clears temporary errors and often fixes scanning problems.
5.  **Scan scope:** If you get zero results, try clearing your current settings and resetting the scan parameters to default.

## 📋 Best Practices

To get the most out of your scanner, follow these habits:

*   Scan during different times of the day to account for network congestion.
*   Test IPs immediately after you update your proxy client.
*   Keep your list of scanned results organized by date.
*   If your speed drops, run a new scan to find a better address.
*   Compare results from different internet service providers if you have more than one connection available.
